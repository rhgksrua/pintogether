# Pintogether

FCC pinterest clone using NodeJS, ExpressJS, and ReactJS.
mongoDB for database.

## Getting Started

```
npm install
```

Environment variables are required.  This project uses `dotenv`.

Required env var:
* `GITHUB_CLIENT_ID`
* `GITHUB_CLIENT_SECRET`
* `CALLBACK_URL`

```
npm run dev:watch
```

Tests...
```
npm test
```
